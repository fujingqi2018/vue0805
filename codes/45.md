# vue 的动画之过渡动作

## 笔记

`transition`标签包裹
该标签的 name 属性用于指定 style 中的类名，比如`name="fjq"`
则

```css
/* 进入 和 离开了 动画 */
.fjq-enter,
.fjq-leave-to {
  ...;
}
/* 进入了 和 离开 动画 */
.fjq-enter-to,
.fjq-leave {
  ...;
}
/* 进入过程中 和 离开过程中 动画 */
.fjq-enter-active,
.fjq-leave-active {
  ...;
}
```
